🤖 リアルタイム音声対話AIアシスタント

Claude + OpenAI を活用したマルチLLM音声アシスタント

📦 GitHub で見る ⬇️ ダウンロード
📌 デモについて: このアプリケーションはバックエンドサーバーが必要なため、 ローカル環境でのセットアップが必要です。以下の手順に従ってください。

🎤 音声モード

Google Cloud Speech-to-Text/Text-to-Speechを使用したリアルタイム音声対話

📝 テキストモード

ブラウザベースのチャットインターフェースでAIと対話

⚡ マルチLLM

Claude APIとOpenAI APIを並列実行し、最速応答を採用

🛡️ 高信頼性

片方のAPIがエラーでも自動フォールバックで継続動作

🚀 クイックスタート

  1. リポジトリをクローン:
    git clone https://github.com/ougonnokishi/voice-assistant.git
  2. 依存関係をインストール:
    cd voice-assistant && npm install
  3. 環境変数を設定:
    cp .env.example .env
    （APIキーを設定）
  4. サーバーを起動:
    npm start
  5. ブラウザでアクセス:
    テキストモード: http://localhost:3000/text-mode.html
    音声モード: http://localhost:3000

必要なAPIキー

  • ANTHROPIC_API_KEY - Anthropic Console
  • OPENAI_API_KEY - OpenAI Platform
  • Google Cloud認証 - 音声機能用（オプション）
